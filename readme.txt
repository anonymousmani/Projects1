Show weather forecast report in png format and data is also available in mongodb database
in JSON format......\

I am using Python 3.6 IDLE environment for coding. Following libraries will be used in the app.

Pymongo – For interfacing Python with MongoDB
Threading – For creating parallel threads
Json – For handling JSON data coming from APIs
Httplib2 – For making http request to download API data
Folium – For creating Map
Seaborn – For plotting forecast data
Geopy – For getting geographical coordinates of a location
To install any of the above libraries, simply open Command Prompt and install using pip install
